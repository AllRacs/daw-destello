# Spark web

Social network similar to instagram.

What can you do?
* Search images by image name, user or album
* Register and login
* Create albums
* Upload images adding them to an album
* Like a post
* Modify your account
* Delete your account
* Chosse a theme

Setup:
1. Install XAMPP (or similar)
2. Clone repo in [...]
3. Run XAMPP
4. ...
